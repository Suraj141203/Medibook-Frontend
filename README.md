💊 Medibook – Medicine Booking Application

Medibook is a modern medicine booking web application built using Angular.
The platform allows users to search medicines, add them to a cart, and place orders online easily through a simple and responsive interface.

This project aims to simplify the process of purchasing medicines by providing a digital solution for online medicine booking.

🚀 Features

🔐 User Authentication (Login / Registration)

💊 Browse available medicines

🔎 Search medicines easily

🛒 Add medicines to cart

📦 Place medicine orders

📜 View order history

📱 Responsive user interface

🔗 REST API integration with backend services
